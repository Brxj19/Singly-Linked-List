Overview

`SinglyLinkedList` is a modern, header-only C++17 library that provides a singly linked list container, similar in functionality to `std::forward_list`. It is built with modern C++ practices, including smart pointers for automatic memory management (`std::unique_ptr`), move semantics, and emplacement for optimal performance.

This library is ideal for scenarios where you need efficient insertions and deletions at the beginning of a sequence and don't require bidirectional iteration.

Features

  • Automatic Memory Management: No manual `new` or `delete` required.
  • Full Support for Move Semantics: Efficiently move objects instead of copying them.
  • In-place Construction: `emplace` functions construct objects directly in the list, avoiding temporary copies.
  • STL-like Interface: Follows standard library conventions for methods like `push_front`, `pop_front`, `begin`, `end`, etc.
  • Full Iterator Support: Provides `iterator` and `const_iterator` for easy traversal and compatibility with STL algorithms.
  • Exception Safe: Throws standard exceptions like `std::out_of_range` for invalid operations.

Getting Started

Prerequisites

A C++ compiler that supports the C++17 standard or newer (e.g., GCC 7+, Clang 5+).

Compilation

Since this is a header-only library, there is no need to build it separately. Simply include `SinglyLinkedList.h` in your project and compile your source files with the C++17 standard enabled.

Example compilation command:

g++ -std=c++17 your_main_file.cpp -o your_program

Basic Usage

#include <iostream>
#include "SinglyLinkedList.h"

int main() {
    // Create a list using an initializer list
    SinglyLinkedList<int> list = {10, 20, 30};

    // Add elements
    list.push_front(5);  // list is now {5, 10, 20, 30}
    list.push_back(40); // list is now {5, 10, 20, 30, 40}

    // Iterate and print elements
    std::cout << "List contents: ";
    for (const auto& val : list) {
        std::cout << val << " ";
    }
    std::cout << std::endl;

    // Access elements
    std::cout << "Front: " << list.front() << std::endl;
    std::cout << "Back: " << list.back() << std::endl;

    // Remove an element
    list.pop_front(); // list is now {10, 20, 30, 40}

    return 0;
}

Usage Examples

Emplacement

Emplacement constructs objects directly within the list, avoiding temporary objects. This is highly efficient for complex types.

struct Record {
    std::string name;
    int id;
    Record(const std::string& n, int i) : name(n), id(i) {
        std::cout << "Record created!" << std::endl;
    }
};

SinglyLinkedList<Record> records;
// Constructs a Record in-place. No temporary Record object is created.
records.emplace_front("Alice", 101);

Iterators and `insert_after`/`erase_after`

Iterators are used to specify positions for insertion and erasure.

SinglyLinkedList<int> list = {10, 20, 40, 50};

// Get an iterator to the second element (20)
auto it = list.begin();
++it; 

// Insert 30 after 20
list.insert_after(it, 30); // list is now {10, 20, 30, 40, 50}

// Erase the element after 30 (which is 40)
++it; // it now points to 30
list.erase_after(it); // list is now {10, 20, 30, 50}

Reversing the List

SinglyLinkedList<char> list = {'A', 'B', 'C', 'D'};
list.reverse(); // list is now {'D', 'C', 'B', 'A'}
